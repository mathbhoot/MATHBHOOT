# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability related to MATHBHOOT, please do not disclose it publicly.

Use the repository's private [GitHub Security Advisory form](https://github.com/mathbhoot/MATHBHOOT/security/advisories/new) and include:

- A description of the issue
- Steps to reproduce the problem
- Potential impact
- A suggested fix, if available

Reports will be reviewed and addressed as appropriate. The machine-readable disclosure policy is available at `https://mathbhoot.com/.well-known/security.txt`.

## Supported Versions

MATHBHOOT is currently under active development. Security updates are applied to the latest version available at `https://mathbhoot.com`.

## Scope

This policy applies to:

- MATHBHOOT website
- Repository source code
- Project infrastructure
- Future web applications and services associated with MATHBHOOT

## Responsible Disclosure

Please allow reasonable time for investigation and remediation before publicly disclosing any issue. Do not access, modify, retain, or distribute other people's data while testing.
